# Blender Add-on: Apply Selected Modifier

## Overview
This Blender add-on provides a simple functionality to apply active modifiers to selected objects directly from the Modifier Properties panel. It adds a button labeled "Apply active modifiers to selected" above the "Add Modifier" button.

## Features
- Apply active modifiers to all selected objects in the scene.
- Easy integration into the Modifier Properties panel.

## Installation
1. Download the add-on files.
2. Open Blender and go to `Edit` > `Preferences`.
3. In the Preferences window, navigate to the `Add-ons` tab.
4. Click on `Install...` and select the `apply_selected_modifier.py` file.
5. Enable the add-on by checking the box next to its name.

## Usage
- Select the objects you want to apply the active modifiers to.
- Go to the Modifier Properties panel.
- Click the "Apply active modifiers to selected" button to apply the modifiers.
